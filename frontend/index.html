AdLoop

The AI creative team that critiques its own work

AdLoop grades its own ads before you ever see them.

Paste a URL and a society of Qwen-powered agents researches your brand, writes framework-diverse copy, grounds each design in real Pinterest references, and ships finished on-brand creatives. A vision-model critic scores every one from 0 to 100 and sends weak work back for revision, so only ads that clear the bar reach you. Memory means each run beats the last. Built for the Qwen Cloud Hackathon, Track 3: Agent Society.

A society of specialised agents

Each agent owns one job and hands off over a shared blackboard. The Critic scores every creative and sends weak work back for revision. That self-critique loop is the moat.

How a run works

  1. Strategist scrapes and web-searches the brand into a structured Brand Kit.
  2. Director plans the campaign and orchestrates the sub-agents.
  3. Copywriter writes several framework-diverse angles, each anchored to a real fact.
  4. Design Researcher pulls real Pinterest references into a design brief.
  5. Art Director generates each creative and composites the type layer.
  6. Critic scores every creative 0 to 100 and sends weak work back for revision.
  7. The best result is kept, and lessons are written to memory for next time.

Everything runs on Qwen Cloud. The self-critique loop and shared memory are what make this a Track 3 agent society, not a single prompt.

About AdLoop

AdLoop is an open-source, multi-agent "society" that turns a single product, service, or app URL into finished, on-brand ad creatives. Built for the Global AI Hackathon with Qwen Cloud, Track 3: Agent Society.

A Strategist scrapes and web-searches the brand into a structured Brand Kit. A Director orchestrates a Copywriter, a Design Researcher (grounded in real Pinterest references), and an Art Director. A vision-model Critic scores every creative 0 to 100 and sends weak work back for revision before you see it, and a persistent memory makes each run sharper than the last.

Everything runs on Qwen Cloud (DashScope): qwen-plus, qwen-max, qwen-vl-max, wan2.2 for images, and text-embedding-v3 for memory.
